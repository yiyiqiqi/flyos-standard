# flyOS 移植层说明

[English](README.md)

本目录保存 `flyos-standard` 中的端口边界代码。

## 范围

当前移植层负责：

- 上下文切换支持；
- Tick 与中断相关的底层钩子；
- 平台相关的运行时 glue 代码；
- kernel 代码与板级 HAL/SDK 代码之间的边界。

## 当前支持平台

- `stm32f4`
- `s32k344`

## 试用者可以期待什么

试用者应当能够：

- 找到平台相关的端口文件；
- 理解哪些底层职责仍应由板级工程承担；
- 看到 `FLYOS_PLATFORM` 如何选择目标实现。

## 这一层当前不承诺什么

当前公开移植层并不承诺所有下游板级集成都是开箱即用的。用户仍然需要有效的启动环境、链接脚本、时钟配置和 HAL/SDK 集成。
